# a2mgaps

## Requirements

- numpy
- biopython
- matplotlib

## Usage

    a2mgaps <outpath.pdf> <inpath.a2m> [<inpath.a2m> ...] [OPTIONS]

    --allcols             Include all columns in the plot (also lower case
                          characters)
    --cpos CPOS [CPOS ...]
                          Concatenation position(s) for concatenated MSAs.
